# EnigmaApp — 專案定義

## 專案概述

EnigmaApp 是一款模擬二戰期間德軍 Enigma 密碼機的 macOS 原生應用程式。使用者可透過逼真的 3D 互動介面操作虛擬 Enigma 機器，體驗歷史上的加密與解密流程。

## 核心目標

- 忠實重現 Enigma 密碼機的外觀與運作機制
- 提供可互動的 3D 模擬操作體驗
- 利用 Mac GPU 加速 3D 渲染，確保流暢的視覺效果

## 技術棧

| 類別 | 技術選型 |
|------|---------|

## 領域知識

### Enigma 密碼機簡介

Enigma 是二戰期間德軍廣泛使用的電動機械式加密裝置，主要由以下元件組成：

- **鍵盤（Keyboard）**：26 個字母按鍵，用於輸入明文
- **燈板（Lampboard）**：26 個燈泡，顯示加密後的字母
- **轉子（Rotors）**：通常 3 個（可選自 5 個），每個轉子定義一組字母替換對應，每次按鍵後轉子會旋轉
- **反射器（Reflector）**：將訊號反射回轉子，確保加密與解密使用相同設定
- **接線板（Plugboard）**：可交換最多 13 組字母對，增加加密複雜度

### 加密流程

1. 操作員按下鍵盤上的字母
2. 訊號依序通過接線板 → 轉子（由右至左）→ 反射器 → 轉子（由左至右）→ 接線板
3. 燈板上對應的字母亮起，即為加密結果
4. 最右側轉子旋轉一格；達到特定位置時帶動下一個轉子旋轉（類似里程表機制）
