# Spec: macOS 應用程式架構

## 概述

定義 EnigmaApp 的 macOS 原生應用程式架構，包含視窗管理、選單列、設定介面與整體 UI 佈局。

## 功能需求

### 應用程式視窗

- 主視窗顯示 3D Enigma 密碼機場景
- 支援視窗縮放，3D 場景自適應大小
- 支援全螢幕模式

### 設定面板

- **轉子選擇**：從 5 種轉子中選擇 3 個並設定順序
- **轉子初始位置**：設定每個轉子的起始字母
- **環設定**：設定每個轉子的 Ring Setting
- **反射器選擇**：選擇 Reflector B 或 C
- **接線板設定**：圖形化介面設定字母對交換
- 設定面板可作為側邊欄或獨立面板呈現
- 設定面板在視窗縮放時須維持穩定佈局，Picker 與 TextField 不得過度拉伸

### 操作面板

- 文字輸入區：可直接輸入明文進行批次加密
- 輸出顯示區：顯示加密/解密結果
- 重置按鈕：將轉子位置重置為初始設定，同時清除輸入文字、輸出文字與操作歷程記錄
- 歷史記錄：顯示本次操作的輸入/輸出歷程

### 選單列

- 檔案選單：匯出/匯入機器設定
- 編輯選單：複製加密結果
- 檢視選單：切換設定面板顯示、重置視角
- 說明選單：Enigma 密碼機歷史介紹、操作說明

### 鍵盤快捷鍵

- 支援實體鍵盤輸入（A–Z），同步觸發 3D 鍵盤按壓動畫與加密
- ⌘R：重置轉子位置
- ⌘E：匯出設定
- ⌘I：匯入設定

## 驗證標準

- 應用程式可在 macOS 14+ 正常啟動
- 視窗縮放與全螢幕模式正常運作
- 實體鍵盤輸入與 3D 操作結果一致
- 機器設定可正確匯出與匯入
